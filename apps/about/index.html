The MMCD Analytics Platform is a comprehensive suite of analytical tools designed to support the Metropolitan Mosquito Control District's operations, surveillance, and treatment activities. This platform provides real-time insights and historical analysis capabilities for mosquito control professionals.

Technology Stack

  • R & Shiny: Interactive web applications and data analysis
  • PostgreSQL: Enterprise database for operational data
  • Leaflet: Interactive maps and geospatial visualization
  • Plotly & ggplot2: Advanced charting and visualization
  • Docker: Containerized deployment and scalability
  • HTML/CSS/JavaScript: Frontend interface and user experience

Platform Features

  • Real-time Data: Live connections to operational databases
  • Interactive Visualization: Dynamic charts, maps, and tables
  • Historical Analysis: Trend analysis and pattern recognition
  • Responsive Design: Optimized for desktop and mobile devices
  • Modular Architecture: Scalable and maintainable codebase

Supervision

This project was supervised by Nancy Read, Data Systems Coordinator at the Metropolitan Mosquito Control District (MMCD). Her guidance and expertise were instrumental in shaping the platform's development and ensuring its alignment with MMCD's operational goals.
